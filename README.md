KursV-master(одногруппника) - отправляет актуальные курсы валют моему API в структурированном JSON-формате.

Convertation(моё) - конвертирует актуальные курсы в структурированном JSON-формате.

currency-converter-frontend - одностраничное vite-react приложение, который предоставляет веб-интерфейс для двух API.

Логика работы веб-сервиса - пользователь открывает интуитивно понятный веб-интерфейс, где видит простую форму с полями с вводом суммы и выбором валют. Также он видит две кнопки для обновления курсов валют и обновлением истории конвертации. После ввода суммы конвертации и выбрав интересующие валюты система автоматически актуализирует курсы валют, потом рассчитывает по этому курсу нашу валюту и полученный результат отображается в форме в веб-интерфейсе. Весь процесс происходит в фоновом режиме — пользователь просто получает конвертированную по курсу валюту в удобном формате без необходимости переходить между разными сервисами или вручную обрабатывать данные.
